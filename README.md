# 12 Tablas Digital — Marketplace de plugins

Plugin **12-tablas-laboral**: 15 skills para estudios laborales argentinos (parte trabajadora).

## Instalar (Cowork)
Plugins -> Agregar marketplace desde GitHub -> URL de este repo -> Instalar **12-tablas-laboral**.

## Actualizar
Cada release sube el campo `version` en `plugins/12-tablas-laboral/.claude-plugin/plugin.json`. El cliente ve el boton "Actualizar".
